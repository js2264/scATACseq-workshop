2. Prerequisites and required machine configuration

Prerequisites

The course is intended for those who have basic familiarity with Unix and the R scripting language.

  • If a refresher is needed for Unix command line (hopefully not), please go over this tutorial and its companion cheatsheet.
  • Getting down to basics: an introduction to the fundamentals of R (courtesy of Mark Ravinet).
  • Gentle introduction to R/Biocondutor: here
  • For a full in-depth guide of Bioconductor ecosystem: read the comprehensive R/Bioconductor book from Kasper D. Hansen available under the CC BY-NC-SA 4.0 license [PDF]

Local configuration

  • Install R and other important softwares
conda create --name scatac
conda activate scatac
conda install -c conda-forge -c bioconda \
    r-base r-essentials r-devtools r-tidyverse r-biocmanager r-polyclip r-rcpparmadillo \
    trim-galore bowtie2 samtools deeptools meme macs2 igv sinto yapc htseq subread ucsc-bedgraphtobigwig bedtools
conda install -c anaconda hdf5
  • The following R packages have also been installed (along with their many dependencies, of course!):
